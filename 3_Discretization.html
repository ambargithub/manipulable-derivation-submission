Discretization

Rebuilding the ideal system using small elements

We have now gained some intuition about the behavior of this string. But can we construct this string using simpler elements?
For instance, as the behavior of a collection of elements, all of which behave the same way?
In order to find that out, we need to introduce an important assumption in our derivation. We assume that our string is composed of numerous small mass points.

This step is called discretization. When the string is unstretched, the distance between any two mass points is dx, which tends to zero. ('tends to' roughly means 'is close to')
